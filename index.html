Simple Check-In

  • Step 1

    Display PNR

    *823331JANSFOGDL-SMITH or

    *-SMITH or

    *QFFY4C

    Review PNR with passenger to determine final destination. If passenger is connecting to another carrier and it is not in the PNR you will need to add a 'BK' segment for the OAL space (e.g.: 0AZ202Y22SEPFCOLHRBK1). This will allow you to check baggage to the final destination.

    Step 2

    Display Electronic Ticket

    ETR*

    Examine the passenger ticket for: Correct flight date, Reservation status, Class of service and Fare.

    Step 3

    Check for a valid passport

    Items to verify:
    Surname, Given Name, Photo, Date of Expiration, Passport Number, Place of Issue, Nationality.


    Viewing TIMATIC in DELANET:
    Click on INTL Documentation
    Located under ACS Tools
    Click on TIMATIC

    Step 4

    Assign seat if not reserved in PNR

    A*C2

    Display seat map for segment 2. '-' displayed on a the seat map indicates the seat is available to assign.


    A*$A(Aisle) W (Window) 21A(Specific Seat)

    Example: A*$A A*$W A*$21A


    A*S2(Segment 2) $ /A /W/21A

    Example: A*S2$A A*S2$W A*S2$21A

    Step 5

    Add SkyMiles account number of not already entered

    4SSRFQTVDLHK/AM0123456789-KOONTZ/LORI

    Replace AM with the applicable OAL 2-letter code.

    Step 6

    Travel Documents Template

    PS*

    Scan passport or entry passenger passport info manually.

    PS*1.1 passenger No. 1

    PS*INF for an Infant

    Step 7

    Display Passenger Contact Template

    PC*

    If passenger is using American passport this template will will load automatically tab to "INFORMATION DECLINED/?N" and type "Y".

    If passenger is using Mexican passport you can omit this step.

    Step 8

    Print Bag Tags

    PBT*

    AeroMexico bag allowence is 1 bag 50lb.

    on the template enter 1 and then enter p to print the bag tags

    Step 9

    Print Boarding Pass

    A*B

    Print boarding pass and advise passenger of gate number and location.

Upgade Seats

  • Upgade Seats

  • Step 1

    Verify that there are available seats in First Class cabin

    You must ask Prime Agent or AeroMexico representative for availability even if the Seat Map shows empty seats.

    Step 2

    Upgrade Charges

    MDMCO

    Display MDMCO template and charge $150.00 for Upgrade.

    Step 3

    Assign First Class Seat

    To assign seat you must be Prime or ask Prime agent to assign the seat for you.

    A*S1$<Seat #>IVJ

    If you took Primary to do the above entry using

    AQRA-P

    Be sure you release primary after the entry using

    AQRD-P

    Step 4

    Start a regular check-in process

    Upgraded passengers have two 70lb bags allowance but you have to override the template (pre-paid) because the system will trigger the overweight template over 50lb bag.

    Step 5

    No Meal

    Passenger has to sign the No Meal Waiver Form.

    Unaccompanied Minor Customer Service Request, form.

    Customs form.

    If the UMNR is traveling with American passport, the Immigration form.
